Create Grid from Class

Input a C# POCO (public properties only) and generate a baseline XAML grid. The default control type is TextBox. DateTimes will generate a DatePicker, and bools will generate a CheckBox, but for other data types you will need to manually change the control type. The grid rows are created in the order that they appear in the class. Row and Column numbering starts at 1, based on my personal preference to use the first row and column as a margin.
