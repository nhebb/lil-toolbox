Backing Props Generator Caliburn.Micro

Creates the backing properties for binding controls in a given XAML input to the View Model, for use with Caliburn.Micro. It makes assumptions about the appropriate data type based on the control type. Resulting code should be audited / edited after generation. This is not an XAML parser, so the x:Name element must be on the same line as the control type, e.g. <DatePicker x:Name="BirthDay" ...
